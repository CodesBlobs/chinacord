Frontend-only voice stack

DropZone Voice

Lightweight gaming voice rooms powered entirely by the browser. Create a room, join from another tab or window, and talk with WebRTC. Supports multinational calls now. Created by Sir Egghead the Third(fred in his Egghead arc)

This frontend-only build uses BroadcastChannel and localStorage for room state and signaling, so it works across tabs and windows on the same browser profile and origin.
